# AWS Serverless Notes App

## Overview

A serverless Notes Application built using AWS Cloud services.

Users can:

* Create Notes
* View Notes

## Technologies Used

* Amazon S3
* Amazon API Gateway
* AWS Lambda
* Amazon DynamoDB
* HTML
* JavaScript

## Architecture

User
→ S3 Website
→ API Gateway
→ Lambda
→ DynamoDB

## AWS Resources

### S3 Bucket

isha-aws-notes-app-2026

### Lambda Function

notes-handler

### DynamoDB Table

NotesTable

### API Gateway Routes

GET /notes

POST /notes

## Features

* Add Notes
* View Notes
* Serverless Architecture
* Cloud Database Storage
